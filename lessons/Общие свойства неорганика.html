Тема 4

Оксиды
Получение и химические свойства

Оксиды — бинарные соединения, содержащие кислород со степенью окисления −2. Атомы кислорода связаны только с атомами другого элемента (нет пероксидной связи −O−O−).

Часть 1 · Классификация

Виды оксидов

Все оксиды делятся на солеобразующие и несолеобразующие.

Несолеобразующие оксиды
N₂O, NO, CO, SiO — им нельзя поставить в соответствие ни кислоту, ни основание.

Солеобразующие делятся на три группы:

Основные
Оксиды металлов со степенью окисления +1 и +2 (кроме ZnO, BeO и нескольких других).
Примеры: Na₂O, FeO, CaO, BaO, Li₂O, K₂O
Амфотерные
Оксиды металлов со степенями окисления +3 и +4 (реже +2: ZnO, BeO).
Запомни: ZnO, BeO, Al₂O₃, Cr₂O₃, Fe₂O₃
Кислотные
Оксиды неметаллов (не относящихся к несолеобразующим) и оксиды металлов в высоких степенях окисления +5, +6, +7.
Примеры: CO₂, CrO₃, Cl₂O₇, SO₃, P₂O₅
🔑
Алгоритм классификации Неметалл? → кислотный или несолеобразующий (проверь список). Металл? → смотри степень окисления: +1, +2 = основный; +3, +4 = амфотерный; +5, +6, +7 = кислотный. Исключения: ZnO и BeO — амфотерные, хотя степень +2.
Часть 2 · Оксид ↔ Гидроксид

Каждому солеобразующему оксиду соответствует гидроксид

Центральный элемент имеет одну и ту же степень окисления в паре «оксид ↔ гидроксид».

Основный оксид → Основание
Na₂O → NaOH
CaO → Ca(OH)₂
CrO → Cr(OH)₂
Кислотный оксид → Кислота
SO₂ + H₂O → H₂SO₃
SO₃ + H₂O → H₂SO₄
CrO₃ + H₂O → H₂CrO₄
P₂O₅ + H₂O → сначала HPO₃, затем H₃PO₄
Амфотерный оксид → Амфотерный гидроксид
ZnO → Zn(OH)₂   |   Al₂O₃ → Al(OH)₃   |   Cr₂O₃ → Cr(OH)₃   |   Fe₂O₃ → Fe(OH)₃
Особые случаи: NO₂ соответствуют две кислоты: HNO₂ и HNO₃. Fe₃O₄ (или FeO · Fe₂O₃) соответствуют два гидроксида: Fe(OH)₂ и Fe(OH)₃.
Часть 3 · Получение оксидов

Способы получения

[1] Из простых веществ и кислорода
С кислородом реагируют все простые вещества, кроме галогенов, инертных газов и благородных металлов (Au, Ag, Pt).
Металлы: Неметаллы: 4Li + O₂ = 2Li₂O H₂ + O₂ → нет (нужна искра!) 2Zn + O₂ = 2ZnO C + O₂ = CO₂ 4Al + 3O₂ = 2Al₂O₃ S + O₂ = SO₂ ⚠ Na + O₂ = Na₂O₂ (пероксид!) 4P + 5O₂ = 2P₂O₅ ⚠ Fe + O₂ = Fe₃O₄ (не Fe₂O₃!) N₂ + O₂ ⇌ 2NO (только при ⚡)
[2] Из сложных веществ и кислорода
Сульфиды, карбиды, водородные соединения (PH₃, H₂S, SiH₄…) при сгорании дают смесь оксидов элементов:
CH₄ + 2O₂ = CO₂ + 2H₂O 2H₂S + 3O₂ = 2SO₂ + 2H₂O SiC + 2O₂ = SiO₂ + CO₂ ⚠ 4FeS₂ + 11O₂ = 2Fe₂O₃ + 8SO₂
[3] Разложение сложных веществ
Нерастворимые гидроксиды и некоторые кислоты разлагаются при нагревании:
Ca(OH)₂ = CaO + H₂O (при t°) 2Al(OH)₃ = Al₂O₃ + 3H₂O (при t°) H₂SiO₃ = SiO₂ + H₂O CaCO₃ = CaO + CO₂↑ (при t°)
Часть 4 · Химические свойства

Свойства оксидов — полная таблица

[1] Взаимодействие с водой
Кислотный оксид + вода = кислота (кроме SiO₂!)
Оксид щелочного/щелочноземельного металла + вода = щёлочь
Амфотерные, несолеобразующие оксиды и SiO₂ с водой НЕ реагируют.
CO₂ + H₂O = H₂CO₃ SO₃ + H₂O = H₂SO₄ P₂O₅ + 3H₂O = 2H₃PO₄ CaO + H₂O = Ca(OH)₂ Na₂O + H₂O = 2NaOH BaO + H₂O = Ba(OH)₂ SiO₂ + H₂O ≠ CuO + H₂O ≠
[2] Оксид + оксид (противоположные по свойствам)
Основный + кислотный = соль
Амфотерный + кислотный = соль
Основный (щел./щ.-з.) + амфотерный = соль
Реагируют только CO₂ и SO₃ с оксидами металлов IA и IIA групп.
Na₂O + SO₃ = Na₂SO₄ CuO + N₂O₅ = Cu(NO₃)₂ Li₂O + ZnO = Li₂ZnO₂ (при сплавлении)
[3] Основные и амфотерные оксиды + кислоты
Основный/амфотерный оксид + кислота = соль + вода
Берём сильные кислоты (HCl, H₂SO₄…) или кислоты средней силы (H₃PO₄, HF). Кремниевая кислота с оксидами НЕ реагирует.
CaO + 2HCl = CaCl₂ + H₂O ZnO + H₂SO₄ = ZnSO₄ + H₂O Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O Na₂O + 2HCl = 2NaCl + H₂O
[4] Кислотные и амфотерные оксиды + основания
Кислотный оксид + основание = соль + вода
Со щелочами реагируют все кислотные и амфотерные оксиды.
SO₃ + 2KOH = K₂SO₄ + H₂O CO₂ + 2NaOH = Na₂CO₃ + H₂O ZnO + 2NaOH(сплавл.) = Na₂ZnO₂ + H₂O Al₂O₃ + 2NaOH(сплавл.) = 2NaAlO₂ + H₂O
[5] Восстановление оксидов металлов
Металлы правее Al в ряду активности восстанавливаются из оксидов под действием H₂, C, CO, Al.
CuO + H₂ = Cu + H₂O (при t°) Fe₂O₃ + 2Al = 2Fe + Al₂O₃ (алюмотермия) FeO + C = Fe + CO (при t°) ZnO + CO = Zn + CO₂ (при t°) ⚠ Fe₂O₃ → Fe₃O₄ → FeO → Fe (поcтадийно с H₂ и CO)
[6] Взаимодействие с кислородом
Оксиды с промежуточной степенью окисления элемента могут окисляться кислородом до высшего оксида.
4FeO + O₂ = 2Fe₂O₃ 2SO₂ + O₂ ⇌ 2SO₃ (кат. V₂O₅) 2CO + O₂ = 2CO₂ ⚠ Только NO из оксидов азота реагирует с O₂: 2NO + O₂ = 2NO₂
[7] Сплавление с карбонатами
Амфотерные оксиды с карбонатами щелочных/щелочноземельных металлов при сплавлении дают новую соль и CO₂.
CaCO₃ + SiO₂ = CaSiO₃ + CO₂↑ Na₂CO₃ + ZnO = Na₂ZnO₂ + CO₂↑ ⚠ ZnO + 2NaCl = Na₂O + ZnCl₂ — это НЕВЕРНО! Нельзя «обменять» кислотный остаток на атомы кислорода.
Важно про SiO₂: При высокой температуре SiO₂ реагирует с основными и амфотерными оксидами. Также SiO₂ растворяется в плавиковой кислоте: SiO₂ + 4HF = SiF₄ + 2H₂O.
Тема 5.1

Основания
Классификация, свойства и получение

Гидроксиды — сложные вещества, содержащие одну или несколько гидроксогрупп (OH). Они делятся на основания, амфотерные гидроксиды и кислородсодержащие кислоты.

Часть 1 · Классификация

Три группы гидроксидов

Основания
Гидроксиды металлов со ст. ок. +1 и +2 (кроме Be(OH)₂, Zn(OH)₂).
Примеры: NaOH, KOH, Fe(OH)₂, Ba(OH)₂, Ca(OH)₂
Амфотерные гидроксиды
Гидроксиды металлов со ст. ок. +3 и +4, а также Be(OH)₂ и Zn(OH)₂.
Запомни: Be(OH)₂, Zn(OH)₂, Al(OH)₃, Cr(OH)₃, Fe(OH)₃
Кислородсодерж. кислоты
Все гидроксиды неметаллов, гидроксиды металлов со ст. ок. +5, +6, +7.
Примеры: HClO, H₂SO₄, HMnO₄, H₂CrO₄

Щёлочи — это гидроксиды щелочных и щелочноземельных металлов (Li, Na, K, Rb, Cs + Ca, Sr, Ba). Они растворимы в воде.

Нерастворимые основания — все остальные (Fe(OH)₂, Cu(OH)₂, Mg(OH)₂…).

⚠️
Не путай! Растворимость ≠ сила электролита. Ca(OH)₂ и Sr(OH)₂ малорастворимы, но являются щелочами, потому что хорошо диссоциируют.

Неустойчивые основания (самопроизвольно разлагаются при попытке получить):

2AgOH = Ag₂O↓ + H₂O (тёмно-коричневый осадок) 2CuOH = Cu₂O↓ + H₂O (кирпично-красный осадок) Hg(OH)₂ = HgO + H₂O (жёлто-оранжевый осадок)
Часть 2 · Химические свойства оснований

Реакции оснований

[1] С кислотами — реакция нейтрализации
Основание + кислота = соль + вода
Для нерастворимых оснований берём сильные кислоты или кислоты средней силы.
2NaOH + H₂SO₄ = Na₂SO₄ + 2H₂O Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O Cu(OH)₂ + 2HNO₃ = Cu(NO₃)₂ + 2H₂O NH₃ + HCl = NH₄Cl (аммиак — бескислородное основание)
[2] С кислотными и амфотерными оксидами
Кислотный/амфотерный оксид + основание = соль + вода
2NaOH + CO₂ = Na₂CO₃ + H₂O 2NaOH + ZnO = Na₂ZnO₂ + H₂O (сплавление) 2KOH + SO₃ = K₂SO₄ + H₂O ⚠ 2NaOH + SiO₂ = Na₂SiO₃ + H₂O (нераств. кремниевая кислота реагирует со щелочами!)
[3] С солями (реакция обмена)
Растворимое основание + растворимая соль = соль + гидроксид
Реакция идёт, если оба реагента растворимы и в продуктах образуется осадок, газ или слабое основание.
2NaOH + FeSO₄ = Na₂SO₄ + Fe(OH)₂↓ (осадок) NH₄Cl + KOH = KCl + NH₃↑ + H₂O (газ) LiOH + KF = LiF↓ + KOH (осадок) ⚠ Ca(OH)₂ + FeS ≠ (FeS нерастворим — реакция не идёт) ⚠ Cu(OH)₂ + K₂CO₃ ≠ (Cu(OH)₂ нерастворим — реакция не идёт)
[4] Термическое разложение
Нерастворимый гидроксид = оксид + вода (при нагревании)
Ca(OH)₂ = CaO + H₂O (при t°) 2Al(OH)₃ = Al₂O₃ + 3H₂O (при t°) Cu(OH)₂ = CuO + H₂O (при t°) ⚠ Щёлочи (NaOH, KOH…) НЕ разлагаются при нагревании!
[5] Окисление Fe(OH)₂
Гидроксид железа(II) легко окисляется на воздухе:
4Fe(OH)₂ + O₂ + 2H₂O = 4Fe(OH)₃ (осадок меняет цвет с серо-зелёного на бурый)
Часть 3 · Получение оснований

Способы получения

[1] Щёлочи: металл/оксид + вода
Щелочной/щ.-з. металл + вода = щёлочь + H₂↑
Оксид щелочного/щ.-з. металла + вода = щёлочь

2Na + 2H₂O = 2NaOH + H₂↑
Na₂O + H₂O = 2NaOH
CaO + H₂O = Ca(OH)₂
[2] Нерастворимые: реакция обмена
Соль + щёлочь = новая соль + гидроксид↓

CuCl₂ + 2NaOH = Cu(OH)₂↓ + 2NaCl
FeSO₄ + 2KOH = Fe(OH)₂↓ + K₂SO₄
⚠ CaCl₂ + NH₃ + H₂O ≠ (Ca(OH)₂ — сильное основание, не осаждается аммиаком!)
Тема 5.2

Кислоты
Классификация и свойства

Кислоты — сложные вещества, которые в растворе диссоциируют на катионы водорода H⁺ и кислотный остаток.

Часть 1 · Классификация кислот

Виды кислот

Сильные кислоты
Бескислородные: HCl, HBr, HI
Кислородсодержащие: HNO₃, H₂SO₄, HClO₃, HClO₄, HMnO₄ и другие, у которых число атомов O и H различается на 2 или 3.
Слабые кислоты
Бескислородные: HF, H₂S
Кислородсодержащие: HNO₂, H₂SO₃, HClO, H₂CO₃, H₃PO₄, H₂SiO₃ и большинство органических кислот.
На ЕГЭ все кислоты делятся только на сильные и слабые. Понятие «кислота средней силы» не применяется.

По основности (числу атомов H, способных замещаться на металл):

Одноосновные: HCl, HBr, HNO₃, HPO₃  |  Двухосновные: H₂SO₄, H₂CO₃, H₂S  |  Трёхосновные: H₃PO₄

⚠️
Неустойчивые кислоты H₂CO₃ → H₂O + CO₂↑ и H₂SO₃ → H₂O + SO₂↑ — разлагаются на оксид и воду. В продуктах ионного обмена записываем как «CO₂ + H₂O», а не H₂CO₃.
Часть 2 · Химические свойства кислот

Реакции кислот

[1] С металлами
Устойчивая растворимая кислота + металл (левее H₂) = соль + H₂↑
Ряд активности: Li Rb K Ba Sr Ca Na Mg Al Mn Zn Cr Fe Cd Co Ni Sn Pb [H₂] Sb Bi Cu Hg Ag Pt Au
Zn + 2HCl = ZnCl₂ + H₂↑ Fe + H₂SO₄(разб.) = FeSO₄ + H₂↑ (получаем соли Fe²⁺!) ⚠ Al + H₂SiO₃ ≠ (нераств. кислота — реакция не идёт!) ⚠ Cu + HCl ≠ (Cu правее H₂) ⚠ Pt + H₂SO₄(разб.) ≠
[2] С основными и амфотерными оксидами
Оксид + кислота = соль + вода (см. раздел «Оксиды»)
CaO + 2HCl = CaCl₂ + H₂O ZnO + H₂SO₄ = ZnSO₄ + H₂O CuO + 2HNO₃ = Cu(NO₃)₂ + H₂O
[3] С основаниями и амфотерными гидроксидами
Кислота + основание = соль + вода (реакция нейтрализации)
NaOH + HCl = NaCl + H₂O Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O NH₃ + HBr = NH₄Br
[4] С солями других кислот
Растворимая кислота + соль (растворимая в кислотах) = кислота + соль
Реакция идёт, если образуется осадок, газ или более слабая кислота.
AgNO₃ + HCl = AgCl↓ + HNO₃ (осадок) CaCO₃ + 2HCl = CaCl₂ + CO₂↑ + H₂O (газ) Na₂SiO₃ + 2HCl = 2NaCl + H₂SiO₃↓ (осадок) ⚠ Более сильная кислота вытесняет более слабую! Ряд: сильные > H₃PO₄, HCOOH, CH₃COOH, H₂SO₃, HF… > H₂CO₃ > H₂S > H₂SiO₃
Кремниевая кислота — самая слабая из изучаемых неорганических кислот. Её можно вытеснить, пропуская CO₂ или H₂S через раствор силикатов: Na₂SiO₃ + CO₂ + H₂O = 2NaHCO₃ + H₂SiO₃↓
Часть 3 · Получение кислот

Способы получения

Кислородсодержащие
Кислотный оксид + вода = кислота
SO₃ + H₂O = H₂SO₄
P₂O₅ + 3H₂O = 2H₃PO₄
⚠ SiO₂ + H₂O ≠ (нельзя получить H₂SiO₃ так!)
Бескислородные
Из простых веществ:
H₂ + Cl₂ = 2HCl
H₂ + S = H₂S
Нелетучая конц. кислота вытесняет летучую из твёрдой соли
NaCl(тв) + H₂SO₄(конц) = NaHSO₄ + HCl↑
NaF(тв) + H₂SO₄(конц) = NaHSO₄ + HF↑
Темы 6.1–6.2

Соли
Классификация, гидролиз, свойства

Соли — сложные вещества, состоящие из атомов металла (или катиона аммония) и кислотного остатка.

Часть 1 · Классификация

Виды солей

ТипОписаниеПримеры
СредниеПолное замещение H в кислоте на металлNaCl, K₂SO₄, Ca₃(PO₄)₂
КислыеНеполное замещение HNaHCO₃, KHSO₄, KH₂PO₄
ОсновныеНеполное замещение OH в основанииMg(OH)Cl, (CuOH)₂CO₃
КомплексныеСложный анион/катион в [ ]Na[Al(OH)₄], K₃[Fe(CN)₆]
ДвойныеДва разных катионаKAl(SO₄)₂
КристаллогидратыМолекулы воды в кристаллической решёткеCuSO₄·5H₂O
К солям НЕ относят: гидриды (NaH, CaH₂), бинарные соединения неметаллов (CS₂, PCl₅), нитриды, фосфиды, карбиды, силициды.

Часть 3 · Химические свойства солей

Реакции солей

[1] С кислотами
Na₂CO₃ + 2HCl = 2NaCl + CO₂↑ + H₂O AgNO₃ + HCl = AgCl↓ + HNO₃ Na₂SiO₃ + 2HCl = 2NaCl + H₂SiO₃↓ ⚠ Сильная кислота вытесняет более слабую
[2] Со щелочами
CuCl₂ + 2NaOH = Cu(OH)₂↓ + 2NaCl NH₄Cl + KOH = KCl + NH₃↑ + H₂O FeSO₄ + 2NaOH = Fe(OH)₂↓ + Na₂SO₄
[3] С солями (обмен)
Р. соль + Р. соль = соль + соль — идёт, если выпадает осадок.
Na₂S + FeSO₄ = Na₂SO₄ + FeS↓ BaCl₂ + K₂SO₄ = BaSO₄↓ + 2KCl ⚠ Cu(NO₃)₂ + KCl ≠ (нет осадка, газа)
[4] С металлами
Металл₁ + растворимая соль₂ = металл₂ + растворимая соль₁
Более активный металл вытесняет менее активный из раствора соли.
Fe + CuSO₄ = Cu↓ + FeSO₄ Zn + 2AgNO₃ = 2Ag↓ + Zn(NO₃)₂ ⚠ Ag + ZnCl₂ ≠ (Ag менее активен, чем Zn) ⚠ Щелочные и щ.-з. металлы НЕ используют — они реагируют с водой!
[5] Разложение при нагревании
Нитраты (продукты зависят от положения металла в ряду активности): Li–Na: 2NaNO₃ = 2NaNO₂ + O₂ (нитрит + O₂) Mg–Cu: 2Cu(NO₃)₂ = 2CuO + 4NO₂ + O₂ (оксид + NO₂ + O₂) Hg–Au: 2AgNO₃ = 2Ag + 2NO₂ + O₂ (металл + NO₂ + O₂) Карбонаты: CaCO₃ = CaO + CO₂↑ Гидрокарбонаты: 2NaHCO₃ = Na₂CO₃ + CO₂ + H₂O Карбонат/гидрокарбонат аммония: NH₄HCO₃ = NH₃ + CO₂ + H₂O
Часть 4 · Кислые соли

Свойства кислых солей

Кислые соли в 90% случаев реагируют по тем же принципам, что и обычные средние соли. Но есть важные особенности:

С одноимённой щёлочью → средняя соль
Щёлочь реагирует с атомом H кислой соли:
NaHCO₃ + NaOH = Na₂CO₃ + H₂O KHSO₄ + KOH = K₂SO₄ + H₂O KH₂PO₄ + 2KOH = K₃PO₄ + 2H₂O
С другой щёлочью
При недостатке щёлочи: NaHCO₃ + Ba(OH)₂ = BaCO₃↓ + Na₂CO₃ + H₂O
При избытке щёлочи: NaHCO₃ + Ba(OH)₂ = BaCO₃↓ + NaOH + H₂O
Получение кислых солей
Средняя соль + одноимённая кислота = кислая соль:
Na₂SO₄ + H₂SO₄ = 2NaHSO₄
Щёлочь + избыток кислотного оксида/кислоты:
NaOH + CO₂(изб.) = NaHCO₃
Особенности гидросульфатов
Растворы NaHSO₄ имеют кислую среду (HSO₄⁻ → SO₄²⁻ + H⁺). Их можно считать смесью Na₂SO₄ и H₂SO₄:
2NaHSO₄ + Mg = Na₂SO₄ + MgSO₄ + H₂↑
2NaHSO₄ + BaCl₂ = Na₂SO₄ + BaSO₄↓ + 2HCl
Тема 8

Амфотерность
Свойства гидроксокомплексов

Амфотерность — способность вещества проявлять и кислотные, и основные свойства. Амфотерные оксиды и гидроксиды образуют бериллий, цинк и металлы в степенях окисления +3 и +4.

Часть 1 · Амфотерные оксиды

Двойственная природа

Кислотные свойства (реагирует как кислотный оксид)
[1] С оксидами щелочных/щ.-з. металлов:
ZnO + Na₂O = Na₂ZnO₂ (при сплавл.)

[2] Со щелочами:
ZnO + 2NaOH(сплавл.) = Na₂ZnO₂ + H₂O
ZnO + 2NaOH(р-р) + H₂O = Na₂[Zn(OH)₄]

[3] Сплавление с карбонатами:
ZnO + Na₂CO₃ = Na₂ZnO₂ + CO₂↑
Основные свойства (реагирует как основный оксид)
[1] С кислотными оксидами:
ZnO + SO₃ = ZnSO₄

[2] С кислотами (если соль существует):
ZnO + H₂SO₄ = ZnSO₄ + H₂O
Al₂O₃ + 6HCl = 2AlCl₃ + 3H₂O
Часть 2 · Амфотерные гидроксиды

Свойства Zn(OH)₂, Al(OH)₃, Cr(OH)₃, Be(OH)₂, Fe(OH)₃

Кислотные свойства
[1] Со щелочами:
Zn(OH)₂ + 2NaOH(сплавл.) = Na₂ZnO₂ + 2H₂O
Zn(OH)₂ + 2NaOH(р-р) = Na₂[Zn(OH)₄]

[2] С оксидами щелочных/щ.-з. металлов:
Zn(OH)₂ + BaO = BaZnO₂ + H₂O
Основные свойства
[1] С кислотами:
Zn(OH)₂ + H₂SO₄ = ZnSO₄ + 2H₂O
Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O

[2] С высшими оксидами (если соль существует):
Zn(OH)₂ + SO₃ = ZnSO₄ + H₂O
Осаждение щелочами: при действии щёлочи на растворы солей Al, Zn, Be, Cr — сначала выпадает осадок амфотерного гидроксида, а при дальнейшем добавлении щёлочи он растворяется:
AlCl₃ + 3NaOH = 3NaCl + Al(OH)₃↓
Al(OH)₃ + NaOH = Na[Al(OH)₄]
Суммарно (избыток щёлочи): AlCl₃ + 4NaOH = 3NaCl + Na[Al(OH)₄]
Часть 3 · Гидроксокомплексы

Комплексные соли амфотерных металлов

При взаимодействии амфотерных гидроксидов с растворами щелочей образуются комплексные соли:

Na[Al(OH)₄] — тетрагидроксоалюминат натрия K₂[Zn(OH)₄] — тетрагидроксоцинкат калия K₃[Cr(OH)₆] — гексагидроксохромат(III) калия

Свойства гидроксокомплексов:

[1] Разложение при нагревании → оксосоли
Na[Al(OH)₄] = NaAlO₂ + 2H₂O (при t°)
Na₂[Zn(OH)₄] = Na₂ZnO₂ + 2H₂O (при t°)
[2] С недостатком сильной кислоты / газами SO₂, CO₂, H₂S
Гидроксокомплекс + кислота (нед.) или газы = соль + амфотерный гидроксид (+ вода)
Na[Al(OH)₄] + HCl(нед.) = NaCl + Al(OH)₃↓ + H₂O Na[Al(OH)₄] + CO₂ = NaHCO₃ + Al(OH)₃↓ K₂[Zn(OH)₄] + 2H₂S = 2KHS + ZnS↓ + 2H₂O ⚠ При пропускании H₂S через тетрагидроксоцинкаты образуются ДВЕ соли + вода
[3] С избытком сильной кислоты → 2 соли + вода
Осадок амфотерного гидроксида полностью растворяется:
Na[Al(OH)₄] + 4HCl(изб.) = NaCl + AlCl₃ + 4H₂O K₂[Zn(OH)₄] + 4HBr(изб.) = 2KBr + ZnBr₂ + 4H₂O
🔑
Особенности Al и Zn Алюминий и цинк реагируют и с растворами, и с расплавами щелочей:
2Al + 2NaOH(р-р) + 6H₂O = 2Na[Al(OH)₄] + 3H₂↑
Zn + 2NaOH(р-р) + 2H₂O = Na₂[Zn(OH)₄] + H₂↑
При сплавлении: 2Al + 6NaOH = 2NaAlO₂ + 2Na₂O + 3H₂↑
Часть 4 · Оксосоли

Na₂ZnO₂, NaAlO₂, NaCrO₂, NaFeO₂

Алюминаты, цинкаты, ферриты и хромиты существуют только в сухом виде. В воде или растворах кислот они разлагаются.

NaAlO₂ + 2H₂O = Na[Al(OH)₄] (гидролиз → комплексная соль) NaFeO₂ + 2H₂O = NaOH + Fe(OH)₃↓ (гидролиз с осадком) NaFeO₂ + 4HCl(изб.) = NaCl + FeCl₃ + 2H₂O (с избытком кислоты)
Для Fe(OH)₃: кислотные свойства проявляются как правило только при сплавлении. Гидроксокомплексы железа в разбавленных растворах НЕ образуются:
Fe(OH)₃ + NaOH(разб.) ≠
Fe(OH)₃ + 3NaOH(конц., кипячение) = Na₃[Fe(OH)₆]

Условия протекания реакций обмена

Когда реакция обмена идёт?

Реакция ионного обмена протекает до конца, если образуется хотя бы один из продуктов:

Реакция ИДЁТ ✓
• Осадок↓ (нерастворимое вещество)
• Газ↑ (CO₂, SO₂, H₂S, NH₃…)
• Вода (слабый электролит)
• Слабый электролит (слабая кислота, NH₃·H₂O)
Реакция НЕ ИДЁТ ✗
• Оба реагента нерастворимы
• Все продукты растворимы и являются сильными электролитами
• Нет осадка, нет газа, нет воды
🔑
Совместный гидролиз Если в растворе встречаются трёхзарядный катион (Al³⁺, Fe³⁺, Cr³⁺) и анион некоторых слабых кислот (CO₃²⁻, HCO₃⁻, S²⁻, HS⁻, SO₃²⁻, HSO₃⁻), реакция идёт необратимо:
3Na₂S + 2AlCl₃ + 6H₂O = 6NaCl + 2Al(OH)₃↓ + 3H₂S↑
