Теория

Закономерности
в таблице Менделеева

Как меняются свойства элементов при движении по периодам и группам. Главное правило: всё стремится либо к фтору, либо к францию.

Главное правило

Фтор и Франций — два полюса

🔵 К фтору (вправо по периоду, вверх по группе):
  • ↑ Неметаллические свойства
  • ↑ Окислительные свойства
  • ↑ Электроотрицательность
  • ↑ Кислотность высших оксидов
  • ↓ Радиус атома
🔴 К францию (влево по периоду, вниз по группе):
  • ↑ Металлические свойства
  • ↑ Восстановительные свойства
  • ↓ Электроотрицательность
  • ↑ Основные свойства
  • ↑ Радиус атома
💡
Мнемоника Чем ближе к F — тем «злее» (отбирает электроны)
Чем ближе к Fr — тем «добрее» (легко отдаёт электроны).
Свойство 1

Металлические и неметаллические свойства

Металлические свойства — блеск, ковкость, пластичность, высокая температура плавления, лёгкость отдачи электронов.

Неметаллические свойства — газообразное или хрупкое твёрдое состояние, лёгкость принятия электронов.

По периоду (→)
Металлические ↓ уменьшаются
Неметаллические ↑ усиливаются
Na → Mg → Al → Si → P → S → Cl
По группе (↓)
Металлические ↑ усиливаются
Неметаллические ↓ уменьшаются
Li → Na → K → Rb → Cs → Fr
Максимум металлических свойств — Франций (Fr). Максимум неметаллических свойств — Фтор (F).
Свойство 2

Окислительные и восстановительные свойства

Окислитель — отбирает электроны. Восстановитель — отдаёт электроны.

Окислительные свойства
↑ усиливаются к фтору
(→ по периоду, ↑ по группе)
Самый сильный окислитель — F₂
Восстановительные свойства
↑ усиливаются к францию
(← по периоду, ↓ по группе)
Самый сильный восстановитель — Fr
N → O → F: окислительные свойства усиливаются
Rb → Cs → Fr: восстановительные свойства усиливаются
Свойство 3

Электроотрицательность (ЭО)

Электроотрицательность — способность атома притягивать к себе электроны в химической связи.

Самый электроотрицательный: F (4,0) > O (3,5) > N (3,0) > Cl (3,0) > Br (2,8)...
Самый электроположительный: Fr → Cs → Rb → K → Na → Li
По периоду (→)
ЭО ↑ увеличивается
Na<Mg<Al<Si<P<S<Cl
По группе (↓)
ЭО ↓ уменьшается
F>Cl>Br>I>At
Свойство 4

Радиус атома

Радиус зависит от числа уровней (по группе) и от «сжатия» ядром (по периоду).

По периоду (→)
Радиус ↓ уменьшается
Новые электроны притягиваются ядром сильнее → атом сжимается
Na > Mg > Al > Si > P > S > Cl
По группе (↓)
Радиус ↑ увеличивается
Появляются новые электронные уровни → атом растёт
Li < Na < K < Rb < Cs < Fr
Внимание: K (радиус больше) vs Cu — в длиннопериодической таблице Cu не в 1-й группе! Радиус K > Cu, хотя в короткой таблице Cu «ниже» K.
Свойство 5

Высшая степень окисления (СО)

Степень окисления — гипотетический заряд атома, если все связи ионные. Обычно высшая СО = номер группы.

По периоду (→)
Высшая СО ↑ увеличивается
Na(+1) Mg(+2) Al(+3) Si(+4) P(+5) S(+6) Cl(+7)
По группе (↕)
Высшая СО = не изменяется
Li Na K Rb Cs — все +1
F Cl Br I — все +7 (кроме F!)
Исключение! У фтора нет положительных СО — он самый электроотрицательный, никогда не отдаёт электроны. Высшая СО фтора = 0 (только в F₂).
Свойство 6

Кислотно-основные свойства высших оксидов и кислот

Высший оксид — оксид элемента в высшей СО. Например, у серы (VI группа) высший оксид — SO₃.

По периоду (→)
Основные свойства оксидов ↓
Кислотные свойства оксидов ↑
Na₂O (осн.) → MgO (осн.) → Al₂O₃ (амф.) → SiO₂ (кисл.) → P₂O₅ → SO₃ → Cl₂O₇
По группе (↕)
Снизу вверх — кислотные свойства ↑
Снизу вверх — основные свойства ↓
BeO (амф.) → MgO → CaO → BaO (основный усиливается ↓)
Кислотность кислородсодержащих кислот по периоду (→ усиливается):
H₃PO₄ < H₂SO₄ < HClO₄
Амфотерные оксиды — на границе металлов и неметаллов: BeO, Al₂O₃, ZnO, Cr₂O₃. Реагируют и с кислотами, и со щёлочами.
Свойство 7

Кислотные свойства водородных соединений

Водородные соединения — соединения элемента с водородом (летучие водородные соединения).

По группе (↓) — УСИЛИВАЮТСЯ
Чем длиннее связь E–H, тем слабее, тем легче отщепить H⁺
HF < HCl < HBr < HI
H₂O < H₂S < H₂Se
По периоду (→) — УСИЛИВАЮТСЯ
Рост ЭО элемента → H⁺ отщепляется легче
NH₃ (осн.) < H₂O (нейтр.) < HF (кисл.)
Важно! HF — слабая кислота (несмотря на высокую ЭО F), потому что связь H–F очень прочная и короткая.
Важное замечание · d-элементы

Ловушка с главной и побочной подгруппами

Закономерности работают только внутри одной подгруппы (только главной или только побочной). Нельзя сравнивать d-элементы с s/p-элементами той же «группы» в короткой таблице!

Пример: F–Cl–Mn–Br все в «VII группе» короткой таблицы.
Но Mn — d-элемент (металл!), его НЕ сравнивают с Cl и Br.
Правильный ряд ЭО: Mn < Br < Cl < F
🗺️
Лайфхак Мысленно разверни таблицу в длинный вариант
В длиннопериодической таблице d-элементы стоят отдельно. Медь и калий — не в одном столбце!

Ловушки ЕГЭ

Типичные ошибки

❌ HF — самая сильная из галогенводородных кислот — НЕТ! HF слабая: связь H–F очень прочная. Правильно: HF < HCl < HBr < HI.
❌ Высшая СО фтора = +7 — НЕТ! F — самый ЭО элемент, не имеет положительных СО. Высшая СО = 0 (F₂).
❌ Cu и K в одной группе → можно сравнивать — НЕТ! Cu — d-элемент побочной подгруппы, в длинной таблице стоит отдельно.
❌ Mn, Cl, Br в VII группе → закономерность одна — НЕТ! Mn — побочная подгруппа, Cl и Br — главная. Сравнивать только внутри одной подгруппы!
❌ Кислотность водородных соед. по группе ↓ уменьшается — НЕТ! По группе вниз кислотность водородных соединений УСИЛИВАЕТСЯ (связь E–H длиннее и слабее).
Два полюса — запомни!

Фтор и Франций

Фтор (F) — «максимум»
✓ Макс. неметаллические св-ва
✓ Макс. электроотрицательность (4,0)
✓ Макс. окислительные св-ва
✓ Мин. радиус атома (среди галогенов)
✗ Нет положительных СО
Франций (Fr) — «минимум»
✓ Макс. металлические св-ва
✓ Макс. восстановительные св-ва
✓ Мин. электроотрицательность
✓ Макс. радиус атома
✓ Макс. основность оксида/гидроксида
Алгоритм решения задания 2

5 шагов

  • 1
    Найди элементы в таблице Менделеева. Определи их период и группу.
  • 2
    Проверь: все элементы в одной подгруппе (главной или побочной)? Смешивать нельзя!
  • 3
    Используй главное правило: к F → неметаллические/кислотные/ЭО, к Fr → металлические/основные.
  • 4
    Если элементы в разных периодах И разных группах — сначала расставь по периодам, потом по группам.
  • 5
    Проверь исключения: F (нет +СО), HF (слабая кислота), d-элементы в короткой таблице.

Важные примеры

Кислотность оксидов (3-й период →):
Na₂O < MgO < Al₂O₃ < SiO₂ < P₂O₅ < SO₃ < Cl₂O₇

Кислотность кислот высших оксидов (3-й период →):
NaOH < Mg(OH)₂ < Al(OH)₃ < H₂SiO₃ < H₃PO₄ < H₂SO₄ < HClO₄

Кислотность HHal (по группе ↓):
HF < HCl < HBr < HI

ЭО важнейших неметаллов:
F(4,0) > O(3,5) > N(3,0) = Cl(3,0) > Br(2,8) > S(2,5) > C(2,5) > H(2,1)
