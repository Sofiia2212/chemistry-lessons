Основы
Строение атомов и типичные степени окисления
Галогены (Hal) — элементы главной подгруппы VII группы. На внешнем уровне содержат 7 электронов.
ЭлементКонфигурация внешнего слояНабор степеней окисленияВысшая валентность
F 2s²2p⁵ −1, 0 I
Cl, Br, I ns²np⁵(nd⁰) −1, 0, +1, +3, +5, +7 VII
⚡
Важно!
У атомов всех галогенов, кроме фтора, возможно распаривание электронов, поэтому они могут проявлять степени окисления +3, +5 и +7. У фтора нет свободных d-орбиталей и он самый электроотрицательный элемент — положительных степеней окисления не бывает.
Раздел 1
Простые вещества
Физические свойства и способы получения галогенов.
Свойство 1 Физические свойства

F₂, Cl₂, Br₂, I₂

Фтор F₂

Жёлтый газ, с удушающим запахом, очень ядовитый.

Хлор Cl₂

Жёлто-зелёный газ, с удушающим запахом, ядовитый, тяжелее воздуха.

Бром Br₂

Тяжёлая тёмно-красная жидкость (н.у.), с неприятным удушливым запахом, ядовитая, легко испаряется.

Иод I₂

Кристаллическое вещество серо-фиолетового цвета, с металлическим блеском. Легко возгоняется, образует фиолетовые пары. Ядовитый, с резким раздражающим запахом.

Свойство 2 Способы получения

В промышленности и в лаборатории

```
[1] В промышленности — электролитический способ

Сырьём для производства галогенов являются соли щелочных металлов (натрия, калия). Фтор можно получить только электролизом расплава солей, остальные галогены — электролизом и растворов, и расплавов.

2KF → 2K + F₂ (в расплаве)
2NaCl + 2H₂O → 2NaOH + H₂ + Cl₂ (электролиз)
[2] В лаборатории

Хлор получают действием концентрированной соляной кислоты на KMnO₄, K₂Cr₂O₇, MnO₂ (окислители за счёт атома металла в степени окисления ≥4) и кислородсодержащие соли хлора:

4HCl(конц.) + MnO₂ = Cl₂ + MnCl₂ + 2H₂O

Бромид- и иодид-анионы являются более сильными восстановителями. Поэтому для получения брома и иода используют не только галогеноводородные кислоты, но и их соли. К списку уже перечисленных окислителей добавляются ещё концентрированные серная и азотная кислоты:

16HBr(конц.) + 2KMnO₄ = 5Br₂ + 2MnBr₂ + 2KBr + 8H₂O
2KI + MnO₂ + 2H₂SO₄ = K₂SO₄ + MnSO₄ + 2H₂O + I₂
🔴
Фтороводород и фториды в подобные реакции не вступают.
```

[2] С металлами

F₂, Cl₂, Br₂ — выбираем высшую СО металла

Фтор, хлор и бром реагируют практически со всеми металлами. Если приходится выбирать степень окисления металла в продукте, то пишем более высокую. Например, +3 для железа:

2Fe + 3Cl₂(изб.) = 2FeCl₃
2Fe + 3Br₂(изб.) = 2FeBr₃
Cu + Br₂ = CuBr₂
```
🧊
Иод — особый случай
Даже с активными металлами реагирует только при нагревании. Реакция с магнием и алюминием идёт при комнатной температуре при катализе водой. Если приходится выбирать степень окисления металла в продукте, пишем более низкую. Например, +2 для железа:
Fe + I₂ = FeI₂ (при t°)
```
[3] С водой

От фтора до иода — разное поведение

```
Фтор окисляет воду

Говорят, что во фторе вода горит:

2F₂ + 2H₂O = 4HF + O₂
Хлорная вода («хлорка»)

Смесь соляной и хлорноватистой кислот:

Cl₂ + H₂O ⇌ HClO + HCl

Хлорноватистая кислота разлагается на свету с выделением активного атомарного кислорода. Это объясняет обеззараживающее и отбеливающее действие хлорной воды:

HClO = HCl + [O]
2Cl₂ + 2H₂O = O₂ + 4HCl (суммарно)
Бром в воде

Обратимо диспропорционирует:

Br₂ + H₂O ⇌ HBrO + HBr
Иод в воде

Практически не растворяется, но растворимость резко повышается в присутствии иодида калия за счёт образования комплекса (жёлто-бурая окраска):

I₂ + KI = K[I₃]
```
[4] Со щелочами

Диспропорционирование — холодные и горячие растворы

Хлор, бром и иод в растворах щелочей диспропорционируют. Продукты реакции зависят от температуры.

```
Холодный раствор (~0°C)

Образуются хлорид и гипохлорит или бромид и гипобромид. Для иода протекание реакции по такой схеме невозможно.

2KOH + Cl₂ = KCl + KClO + H₂O
2NaOH + Br₂ = NaBr + NaBrO + H₂O
Нагретый раствор

Также диспропорционирование, но атомы галогена окисляются до СО +5, а не +1. У иода реакция со щелочами всегда протекает по этой схеме:

6NaOH + 3Cl₂ = 5NaCl + NaClO₃ + 3H₂O (при t°)
6KOH + 3I₂ = 5KI + KIO₃ + 3H₂O (всегда)
💡
Растворы карбонатов калия или натрия имеют щелочную среду из-за гидролиза. Поэтому с ними галогены реагируют похожим образом:
3K₂CO₃ + 3Cl₂ = 5KCl + KClO₃ + 3CO₂ (при t°)
```
[5] Реакции замещения

Более активный галоген вытесняет менее активный

```
Ряд активностиПравило
F₂ → Cl₂ → Br₂ → I₂Окислительная способность галогенов уменьшается. Более активный галоген (стоящий выше) вытесняет менее активный (стоящий ниже) из галогеноводородных кислот и их солей.
2KBr + Cl₂ = 2KCl + Br₂ (появление оранжевой окраски раствора)
2HI + Cl₂ = 2HBr + I₂ (буро окраска; с крахмалом — синее)
I₂ + 5Cl₂ + 6H₂O = 2HIO₃ + 10HCl (при избытке Cl₂ иод окисляется, раствор обесцвечивается)

Галогены также вытесняют серу из сульфидов и сероводорода:

Na₂S + Br₂ = 2NaBr + S↓ (бледно-желтый осадок, раствор обесцвечивается)
H₂S + I₂ = 2HI + S↓
🚫
Примеры реакций, которые не пойдут: NaBr + I₂; CaBr₂ + S. Фтор на практике не используют из-за его слишком высокой окислительной способности.

Хлор проявляет более сильные окислительные свойства, чем бром и иод, поэтому в подходящих условиях может перевести серу в степень окисления +6:

H₂S + 4Cl₂ + 4H₂O = H₂SO₄ + 8HCl
H₂S + Cl₂ = S + 2HCl (если реагируют газы — простое замещение)
```
[6] С другими сложными веществами

ОВР с участием Cl₂ и Br₂

Фтор реагирует со многими сложными веществами, но составлять уравнения этих реакций не придётся. Наиболее распространённые ОВР с участием хлора и брома собраны в таблице. Галогены проявляют окислительные свойства, переходят в низшую степень окисления −1.

```
Переход восстановителяПримеры реакций
S⁺⁴ → S⁺⁶K₂SO₃ + Br₂ + H₂O = K₂SO₄ + 2HBr
SO₂ + Cl₂ + 4KOH = K₂SO₄ + 2KCl + 2H₂O
P⁻³ → P⁺⁵PH₃ + 4Cl₂ + 4H₂O = H₃PO₄ + 8HCl
Fe⁺² → Fe⁺³2FeCl₂ + Cl₂ = 2FeCl₃
Cr⁺³ → Cr⁺⁶3Cl₂ + 2Cr(OH)₃ + 10KOH = 6KCl + 2K₂CrO₄ + 8H₂O
3Br₂ + 2CrBr₃ + 16NaOH = 12NaBr + 2Na₂CrO₄ + 8H₂O

Для иода также характерны восстановительные свойства. Концентрированная азотная кислота и концентрированный раствор перекиси водорода окисляют его до иодноватой кислоты:

I₂ + 10HNO₃(конц.) = 2HIO₃ + 10NO₂ + 4H₂O
I₂ + 5H₂O₂ = 2HIO₃ + 4H₂O
🧪
Степень окисления +5 у иода достаточно устойчива. Иодноватую кислоту можно выделить в чистом виде; при нагревании она разлагается на оксид и воду (без взрыва):
2HIO₃ = I₂O₅ + H₂O (при t°)
```

Свойство 2 Способы получения галогеноводородов

Три метода

```
[1] Синтез из простых веществ

Фтор и хлор реагируют с водородом со взрывом. Реакция образования хлороводорода протекает при нагревании или на свету:

H₂ + Cl₂ = 2HCl

Иод и бром реагируют с водородом менее энергично и только при нагревании. Реакция синтеза иодоводорода обратима:

H₂ + I₂ ⇌ 2HI
[2] Сульфатный способ

При обработке кристаллических фторидов и хлоридов концентрированной серной кислотой выделяются газообразные галогеноводороды:

CaF₂(тв.) + H₂SO₄(конц.) = 2HF↑ + CaSO₄↓
NaCl(тв.) + H₂SO₄(конц.) = NaHSO₄ + HCl↑
⚠️
Этот способ не подходит для получения бромоводорода и иодоводорода. Бромид- и иодид-ионы проявляют более сильные восстановительные свойства, поэтому окисляются концентрированной серной кислотой (возможно образование кислых солей):
2NaBr + 2H₂SO₄(конц.) = Br₂ + SO₂ + Na₂SO₄ + 2H₂O
8KI + 5H₂SO₄(конц.) = 4I₂ + H₂S + 4K₂SO₄ + 4H₂O

Чтобы избежать окисления, можно заменить концентрированную серную кислоту на концентрированную фосфорную:

KI(тв.) + H₃PO₄(конц.) = HI↑ + KH₂PO₄
⚠️
Бромоводород и иодоводород тоже окисляются концентрированной серной кислотой:
2HBr + H₂SO₄(конц.) = Br₂ + SO₂ + 2H₂O
8HI + H₂SO₄(конц.) = 4I₂ + H₂S + 4H₂O
[3] Гидролиз бинарных соединений
PCl₅ + 4H₂O = H₃PO₄ + 5HCl
SiCl₄ + 3H₂O = H₂SiO₃ + 4HCl
```
[1] Общие кислотные свойства

Реакции типичных кислот

Галогеноводороды проявляют свойства типичных кислот. Они реагируют с металлами левее [H₂] в ряду активности, основными и амфотерными оксидами и гидроксидами, солями.

[2] Восстановительные свойства за счёт атома галогена

Cl⁻, Br⁻, I⁻ как восстановители

Фтороводород и фториды не проявляют восстановительные свойства в реакциях, протекающих в растворе. Остальные галогеноводороды или их соли в соответствующих условиях реагируют с типичными окислителями с образованием простых веществ. Важно помнить, что хлорид-ион — слабый восстановитель. Для получения хлора проводят реакцию HCl(конц.) с окислителем за счёт атома металла в степени окисления ≥4 или кислородсодержащей солью хлора.

```
⚠️
Иодидам меди (II) и железа (III) в таблице растворимости соответствует прочерк. Их нельзя получить по реакции обмена или соединения, так как происходит окисление иодид-иона до простого вещества. Металл при этом переходит в соседнюю более низкую степень окисления.
2CuO + 4HI = 2CuI + I₂ + 2H₂O
Fe₂O₃ + 6HI = 2FeI₂ + I₂ + 3H₂O

Аналогичная реакция протекает и при сливании растворов солей (допустимы обе записи):

2CuCl₂ + 4KI = 2CuI + I₂ + 4KCl
2CuCl₂ + 2KI = 2CuCl + I₂ + 2KCl
💡
Нитрат или сульфат меди (I) в продуктах подобных реакций не пишем. Хлорид, бромид и иодид меди (I) нерастворимы в воде. Поэтому признаком реакции, протекающей между растворами солей, будет выпадение осадка и жёлто-бурое окрашивание раствора.
```
[3] Реакция плавиковой кислоты с SiO₂

Травление стекла

Эта реакция используется для получения матового стекла и для нанесения узоров на стеклянные поверхности. Допустимо писать оба уравнения.

SiO₂ + 4HF = SiF₄ + 2H₂O
SiO₂ + 6HF = H₂[SiF₆] + 2H₂O

[1] Термическое разложение хлоратов

Без катализатора и с катализатором — разные продукты

Хлораты щелочных металлов разлагаются на хлорид и перхлорат. Если реакционную смесь продолжить нагревать, то перхлорат разложится на KCl и O₂.

Без катализатора
4KClO₃ = KCl + 3KClO₄ (при t°)
С катализатором MnO₂
2KClO₃ = 2KCl + 3O₂ (при t°, кат. MnO₂)
[2] Участие в ОВР

Сильные окислители за счёт атома хлора

Кислородсодержащие соли и кислоты галогенов являются сильными окислителями (за счёт атома галогена). Если восстановителем является атом этого же галогена в степени окисления −1, то образуется простое вещество по схеме:

```
⚠️
KClO₃ + 6HCl = KCl + 3Cl₂ + 3H₂O
2Cl⁺⁵ + 10e = Cl₂⁰  |  1
2Cl⁻ − 2e = Cl₂⁰  |  5
🎯
В остальных реакциях, которые могут встретиться на ЕГЭ, реализуется переход Cl⁺ⁿ → Cl⁻¹.
5KClO₃ + 6P = 3P₂O₅ + 5KCl (при t°)
3NaClO + 2Cr(OH)₃ + 4KOH = 2K₂CrO₄ + 3NaCl + 5H₂O
```

Применение веществ
  • Хлор — водоочистка, отбеливание, синтез органических растворителей (CCl₄, CHCl₃ и т.д.)

  • Иод — в медицине в виде растворов

Таблица 3 Получение Cl₂, Br₂, I₂ в лаборатории

Какие окислители подходят

Для Cl₂ из HCl(конц.)KMnO₄, K₂Cr₂O₇, MnO₂
Для Br₂, I₂те же + конц. H₂SO₄, конц. HNO₃
HF и фторидыв эти реакции не вступают
Правило Какую степень окисления металла писать

F₂, Cl₂, Br₂ vs I₂

F₂, Cl₂, Br₂ + металлвыбираем более ВЫСОКУЮ СО металла
I₂ + металлвыбираем более НИЗКУЮ СО металла
💡
Пример: Fe + Cl₂(изб.) → FeCl₃ (Fe⁺³), но Fe + I₂ → FeI₂ (Fe⁺²).

Cl₂, Br₂ + горячая щёлочь→ хлорид/бромид + хлорат/бромат (СО +5)

Ловушки ЕГЭ Типичные ошибки
❌ CuI₂ и FeI₃ существуют как обычные соли
✓ Нет! Cu²⁺ и Fe³⁺ окисляют I⁻ до I₂. Образуются CuI (Cu⁺) и FeI₂ (Fe²⁺) + выделяется I₂.
❌ Бромоводород и иодоводород можно получить сульфатным способом (NaBr/NaI + H₂SO₄ конц.)
✓ Нет! Br⁻ и I⁻ окисляются конц. H₂SO₄. Используй конц. H₃PO₄ или другой метод.
❌ Иод реагирует с металлами так же легко, как хлор
✓ Нет! Даже с активными металлами — только при нагревании (с Mg, Al — при катализе водой).
❌ NaBr + I₂ или CaBr₂ + S — реакции идут
✓ Нет! I₂ слабее Br₂ как окислитель — вытеснения не происходит. Проверяй ряд активности.
❌ Высшая СО фтора = +7, как у остальных галогенов
✓ Нет! У фтора нет d-орбиталей и положительных степеней окисления — высшая валентность I, СО только −1 и 0.
Алгоритм Как решать ОВР с галогенами

5 шагов

1
Определи степень окисления галогена в реагенте и реальный диапазон СО (для Cl, Br, I: −1, 0, +1, +3, +5, +7; для F: только −1, 0).
2
Если восстановитель — тот же галоген в СО −1, продукт — простое вещество Hal₂ (например, KClO₃ + HCl → Cl₂).
3
В остальных случаях на ЕГЭ обычно реализуется переход Cl⁺ⁿ → Cl⁻¹ (высшая СО восстанавливается до низшей).
4
Составь электронный баланс, расставь коэффициенты.
5
Проверь сумму зарядов и атомов в левой и правой частях.
