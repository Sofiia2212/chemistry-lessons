Халькогены в таблице Менделеева

Строение атомов, кислород и его соединения, сера, пероксиды, сероводород, сернистый газ и серная кислота. Всё, что нужно для ЕГЭ.

⚗️ Задание 13 · Тема 13
Основы
Строение атомов и степени окисления
Халькогены — элементы главной подгруппы VI группы. На внешнем уровне содержат 6 электронов.
ЭлементКонфигурация внешнего слояСтепени окисленияВысшая валентность
O (Кислород) 2s²2p⁴ −2, −1, 0, +1, +2 II (III в особых случаях)
S (Сера) 3s²3p⁴(3d⁰) −2, −1, 0, +2, +4, +6 VI
⚡
Важно!
У атомов всех халькогенов, кроме кислорода, в возбуждённом состоянии увеличивается число неспаренных электронов — поэтому они могут проявлять степени окисления +4 и +6. У кислорода нет d-орбиталей на 2-м уровне.

Простые вещества
Физические свойства и способы получения кислорода и серы.
Свойство 1 Физические свойства

Кислород, озон и сера

```
Кислород O₂
  • Бесцветный газ, без запаха и вкуса
  • Объёмная доля в воздухе ≈ 21%
  • Поддерживает дыхание, горение, гниение
  • Малорастворим в воде
Озон O₃
  • Светло-голубой газ
  • Нерастворим в воде
  • Характерный запах свежести
  • Применяется для водоочистки
Сера S
Кристаллическая S₈

Жёлтое твёрдое вещество. Нерастворима в воде, растворима в CS₂ и органических растворителях.

Пластическая Sn

Аморфное вещество коричневого цвета. Нерастворима в воде. Получается быстрым охлаждением расплавленной серы.

```
Свойство 2 Получение кислорода

Промышленность и лаборатория

```
В промышленности

Ректификация (фракционная перегонка) сжиженного воздуха. Часть — электролизом воды:

2H₂O → 2H₂ + O₂
На подводных лодках

Регенерация выдыхаемого CO₂:

2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂
В лаборатории — термическое разложение:
а) нитратов: 2KNO₃ = 2KNO₂ + O₂↑
б) хлоратов: 2KClO₃ = 2KCl + 3O₂↑ (кат. MnO₂)
в) перманганатов: 2KMnO₄ = MnO₂ + K₂MnO₄ + O₂↑
г) перекиси: 2H₂O₂ = O₂↑ + 2H₂O (кат. MnO₂)
д) оксидов Hg, Ag: 2HgO = 2Hg + O₂↑
е) карб. серебра: 2Ag₂CO₃ = 4Ag + 2CO₂↑ + O₂↑
🌿
В природе — фотосинтез
6CO₂ + 6H₂O = C₆H₁₂O₆ + 6O₂
```
Свойство 3 Получение серы

Промышленность и лаборатория

В промышленности

Из природных самородков, переработка сульфидных руд, нефти и природного газа.

В лаборатории

Пропускают H₂S и SO₂ через воду — выпадает коллоидный осадок:

2H₂S + SO₂ = 3S↓ + 2H₂O

Химические свойства O₂ и S
Реакции с неметаллами, металлами и сложными веществами.
Раздел 2.1 Взаимодействие с неметаллами

Кислород и сера

```
💡
Кислород реагирует со всеми неметаллами, кроме Cl, Br, I и инертных газов. Реакции (кроме окисления N₂) — экзотермические.
5O₂(изб.) + 4P = 2P₂O₅
O₂ + N₂ ⇌ 2NO − Q (эндотермическая!)
S + O₂ = SO₂
S + H₂ = H₂S
```
Раздел 2.2 Взаимодействие с металлами

Кислород и сера

```
Кислород

Реагирует со всеми металлами, кроме Ag, Au и платиновой группы. Щелочные металлы при комнатной температуре образуют пероксиды (только Li — оксид).

Сера

При комнатной температуре реагирует только с ртутью. Остальные металлы окисляются при нагревании.

3Fe + 2O₂ = Fe₃O₄
2Na + O₂ = Na₂O₂ (пероксид; Li даёт Li₂O)
Fe + S = FeS (при нагревании)
```
Раздел 2.3 Взаимодействие со сложными веществами

Кислород и особые реакции серы

2Cu₂O + O₂ = 4CuO
2Al₂S₃ + 9O₂ = 2Al₂O₃ + 6SO₂
CH₄ + 2O₂ = CO₂ + 2H₂O
Особые реакции серы (с концентрированными кислотами и щёлочью):
РеагентSO₂
+ HNO₃(конц.), t°S + 6HNO₃(конц.) = H₂SO₄ + 6NO₂ + 2H₂O— (не реагирует)
+ H₂SO₄(конц.), t°S + 2H₂SO₄(конц.) = 3SO₂ + 2H₂O— (не реагирует)
+ KOH, t°3S + 6KOH = 2K₂S + K₂SO₃ + 3H₂O (диспропорционирование)— (не реагирует)

Кислород O₂ не вступает в эти реакции — ни с HNO₃(конц.), ни с H₂SO₄(конц.), ни со щёлочью. Это особые реакции именно серы.

Раздел 3
Перекись водорода и пероксиды металлов
Степень окисления O = −1, поэтому проявляют и окислительные, и восстановительные свойства.
Свойство 1 Физические свойства и получение

H₂O₂ — перекись водорода

Свойства

Бесцветная вязкая жидкость. Неограниченно смешивается с водой. Содержит ковалентную неполярную связь O–O.

Получение

Гидролиз пероксидов металлов на холоду:

Na₂O₂ + 2H₂O(хол.) = 2NaOH + H₂O₂
BaO₂ + H₂SO₄(разб.) = BaSO₄ + H₂O₂
💡
Мнемоника
Перекись хранится в холодильнике (= не разлагается на холоду).
Свойство 2 Диспропорционирование

О⁻¹ → О⁰ и О⁻²

2H₂O₂ = 2H₂O + O₂ (свет, t° или MnO₂)
2Na₂O₂ + 2H₂O(горяч.) = 4NaOH + O₂
2Li₂O₂ + 2H₂SO₄(разб.) = 2Li₂SO₄ + O₂ + 2H₂O
2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂
Свойство 3 Окислитель

О⁻¹ → О⁻²

2Fe(OH)₂ + H₂O₂ = 2Fe(OH)₃
2CrCl₃ + 3H₂O₂ + 10NaOH = 2Na₂CrO₄ + 6NaCl + 8H₂O
PbS + 4H₂O₂ = PbSO₄ + 4H₂O (реставрация: чёрный PbS → белый PbSO₄)
Свойство 4 Восстановитель

О⁻¹ → О⁰ (выделяется O₂)

Окислители — Cl₂, KMnO₄, K₂Cr₂O₇, HClO:

2KMnO₄ + 5H₂O₂ + 3H₂SO₄ = 2MnSO₄ + K₂SO₄ + 8H₂O + 5O₂↑
Раздел 4
Сероводород H₂S и сульфиды
Бесцветный газ с запахом тухлых яиц, ядовитый, горючий, малорастворим в воде.
Свойство 1 Кислотные свойства

H₂S — слабая двухосновная кислота

Со щелочами
H₂S + 2NaOH(изб.) = Na₂S + 2H₂O
H₂S + NaOH(нед.) = NaHS + H₂O
«Суровые» сульфиды (чёрный осадок)

С солями Cu, Pb, Ag, Hg — выпадают нерастворимые чёрные осадки:

H₂S + CuSO₄ = CuS↓ + H₂SO₄
🔴
CuS, PbS, Ag₂S нерастворимы в разбавленных кислотах! Растворяются только в конц. HNO₃ или конц. H₂SO₄ при нагревании (ОВР). Это отличает их от обычных растворимых сульфидов (Na₂S), которые с разбавленной кислотой реагируют как обычная соль слабой кислоты:
CuS↓ + H₂SO₄(р-р) ≠ (реакция не идёт)
Na₂S + H₂SO₄(р-р) = H₂S + Na₂SO₄ (для сравнения — обычная соль реагирует)
Свойство 2 Взаимодействие с кислородом

Полное и неполное окисление

Полное окисление (O₂ избыток)
2H₂S + 3O₂(изб.) = 2SO₂ + 2H₂O
Неполное окисление (O₂ недостаток)
2H₂S + O₂(нед.) = 2S + 2H₂O
2CuS + 3O₂ = 2CuO + 2SO₂ (обжиг сульфидов Cu и металлов средней активности)
Свойство 3 Взаимодействие с галогенами

Br₂, I₂ — замещение; Cl₂ — до сульфата

H₂S + Br₂ = S + 2HBr
H₂S + I₂ = S + 2HI
H₂S + 4Cl₂ + 4H₂O = H₂SO₄ + 8HCl (хлор окисляет S²⁻ → SO₄²⁻)
Свойство 4 С сернистым газом и другие ОВР

Реакции с окислителями

2H₂S + SO₂ = 3S↓ + 2H₂O
2KMnO₄ + 3H₂S = 2MnO₂ + 3S + 2KOH + 2H₂O
H₂S + 8HNO₃(конц.,г) = H₂SO₄ + 8NO₂ + 4H₂O
K₂Cr₂O₇ + 3K₂S + 7H₂SO₄ = Cr₂(SO₄)₃ + 4K₂SO₄ + 3S + 7H₂O
Свойство 5 С солями Al, Cr(III), Fe(III)

Совместный гидролиз и ОВР

⚡
Fe(III) → ОВР (сера, FeS не выпадает). Al и Cr(III) → совместный гидролиз с сульфидами.
2FeCl₃ + H₂S = S↓ + 2FeCl₂ + 2HCl
Fe₂(SO₄)₃ + 3(NH₄)₂S = S↓ + 2FeS↓ + 3(NH₄)₂SO₄
3Na₂S + 2AlCl₃ + 6H₂O = 2Al(OH)₃ + 6NaCl + 3H₂S (совм. гидролиз)
Гидролиз Al₂S₃ зависит от среды:
Al₂S₃ + 6H₂O = 2Al(OH)₃ + 3H₂S (нейтральная)
Al₂S₃ + 6HCl = 2AlCl₃ + 3H₂S (кислая)
Al₂S₃ + 8KOH(изб.) = 2K[Al(OH)₄] + 3K₂S (щелочная)
Раздел 5
Сернистый газ SO₂ и сульфиты
Бесцветный газ с резким запахом жжёной серы, ядовитый, растворим в воде.
Свойство 1 Как кислотный оксид

Реакции SO₂ с основаниями и оксидами

H₂O + SO₂ ⇌ H₂SO₃ (обратимая реакция)
2NaOH(изб.) + SO₂ = Na₂SO₃ + H₂O
NaOH(нед.) + SO₂ = NaHSO₃
CaO + SO₂ = CaSO₃
Свойство 2 ОВР — восстановительные свойства

S⁺⁴ → S⁺⁶

2SO₂ + O₂ ⇌ 2SO₃ (кат. V₂O₅ — ключевая реакция производства H₂SO₄!)
5SO₂ + 2KMnO₄ + 2H₂O = 2MnSO₄ + K₂SO₄ + 2H₂SO₄
SO₂ + 2HNO₃(конц.) = H₂SO₄ + 2NO₂
Fe₂(SO₄)₃ + SO₂ + 2H₂O = 2FeSO₄ + 2H₂SO₄ (Fe³⁺ → Fe²⁺)
🔄
В реакции с H₂S сернистый газ — окислитель:
SO₂ + 2H₂S = 3S↓ + 2H₂O
Свойство 3 Особые свойства сульфитов

Совместный гидролиз с Al, Cr(III), Fe(III)

3Na₂SO₃ + 2AlCl₃ + 3H₂O = 2Al(OH)₃ + 6NaCl + 3SO₂ (совм. гидролиз)
2FeCl₃ + Na₂SO₃ + H₂O = 2FeCl₂ + Na₂SO₄ + 2HCl (ОВР с Fe(III))
Свойство 4 Другие ОВР сульфитов

Окисление твёрдых сульфитов и их растворов

Твёрдые сульфиты окисляются кислородом. Реакции с растворами окислителей тоже идут:

2Na₂SO₃ + O₂ = 2Na₂SO₄ (окисление кислородом)
5K₂SO₃ + 2KMnO₄ + 3H₂SO₄ = 6K₂SO₄ + 2MnSO₄ + 3H₂O
K₂SO₃ + Br₂ + H₂O = K₂SO₄ + 2HBr
Na₂SO₃ + 2HNO₃(конц.) = Na₂SO₄ + 2NO₂ + H₂O
Свойство 5 Диспропорционирование и разложение

При нагревании

4Na₂SO₃ = 3Na₂SO₄ + Na₂S (диспропорционирование без доступа воздуха — не для запоминания)
2NaHSO₃ = Na₂SO₃ + H₂O + SO₂ (разложение гидросульфита)
CaSO₃ = CaO + SO₂ (разложение нерастворимого сульфита)
Раздел 6
Серная кислота H₂SO₄
Бесцветная маслянистая ядовитая жидкость. Смешивается с водой с выделением тепла. Сильное водоотнимающее средство.
Свойство 1 Производство

Контактный способ

🏭
FeS₂ → SO₂ → SO₃ (кат. V₂O₅) → олеум → H₂SO₄
```

Ежегодно в мире получают свыше 150 тонн серной кислоты — основное количество расходуют на производство минеральных удобрений и очистку нефти и нефтепродуктов. Также применяется в металлургии, при получении красителей, лекарств, других кислот и солей, как окислитель и водоотнимающее средство.

Сырьё

Пирит FeS₂, самородная сера, серосодержащие газы — отходы цветной металлургии, воздух.

Вспомогательные материалы

Серная кислота (98%), катализатор — оксид ванадия(V).
```

Продукты восстановления

Металлы [Cr…Ag]→ SO₂↑ (чаще всего)

Не реагирует при комнатной температуре с Al, Fe, Cr, Pb (пассивация). Никогда — с Pt, Au.
Ловушки ЕГЭ Типичные ошибки
❌ Суровые сульфиды (CuS, PbS, Ag₂S) растворяются в HCl или H₂SO₄(разб.)
✓ Нет! Нерастворимы в разбавленных кислотах. Растворяются только в конц. HNO₃ или конц. H₂SO₄ при нагревании.
❌ Fe(III) с сульфидами → выпадает FeS
✓ Нет! Идёт ОВР, образуется сера. FeS не пишем — он растворяется в HCl, образующейся в реакции.
❌ Al + конц. H₂SO₄ при комнатной температуре
✓ Пассивация! Реакция идёт только при нагревании, продукт — H₂S.
❌ H₂S + Cl₂ → S + HCl (всегда)
✓ При избытке Cl₂ и воды: H₂S + 4Cl₂ + 4H₂O = H₂SO₄ + 8HCl (S²⁻ → SO₄²⁻).
❌ SO₂ + вода → необратимая реакция
✓ H₂O + SO₂ ⇌ H₂SO₃ — обратима, H₂SO₃ неустойчива.
Алгоритм Как решать ОВР с серой

5 шагов

1
Определи степень окисления серы в реагенте и возможные продукты.
2
Определи силу окислителя: KMnO₄ > K₂Cr₂O₇ > HNO₃(конц.) > Cl₂ > Br₂ > H₂SO₄(конц.).
3
Правило: чем сильнее окислитель → тем выше СО серы в продукте (S²⁻→S⁰→SO₂→SO₄²⁻).
4
Составь электронный баланс, расставь коэффициенты.
5
Проверь сумму зарядов и атомов в левой и правой частях.
