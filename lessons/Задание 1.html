Теория

Электронная конфигурация —
строим атом по правилам

Как расположены электроны в атоме. Главное: знать порядок заполнения орбиталей и уметь его применять.

Основы

Из чего состоит атом

🔵 Ядро
Протоны (заряд +1, масса 1) — определяют, какой это элемент.

Нейтроны (заряд 0, масса 1) — не влияют на химические свойства.
🔴 Оболочка
Электроны (заряд −1, масса ≈ 0) — определяют связи.

В нейтральном атоме: e⁻ = протонам = Z
🧮
Формулы для расчёта Протоны = Электроны = Z (порядковый номер)
Нейтроны = A − Z, где A — округлённая атомная масса
⚠️ Масса хлора Cl = 35,5 — не округляется! Для ³⁵Cl нейтроны = 18.
Изотопы

Атомы одного элемента с разным числом нейтронов. Химические свойства одинаковы — разная только масса.

¹H — 0 нейтронов · ²H (дейтерий) — 1 нейтрон · ³H (тритий) — 2 нейтрона
Все трое — водород. Все образуют «воду»: H₂O / D₂O / T₂O
Уровни и подуровни

Орбитали и их вместимость

Электроны располагаются на энергетических уровнях (1, 2, 3, 4...), каждый делится на орбитали (подуровни).

Ключевое правило

Порядок заполнения орбиталей

Главная последовательность
1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p
d запаздывает! 3d заполняется после 4s — на период позже, чем кажется. Смотри на цвет ячейки в таблице Менделеева.
Алгоритм составления конфигурации
  • 1
    Найди элемент в таблице → определи Z (порядковый номер)
  • 2
    Определи цвет ячейки: розовый → s, жёлтый → p, синий → d
  • 3
    Посчитай, какой по счёту элемент в своём блоке — столько электронов на последней орбитали
  • 4
    Все предыдущие орбитали заполнены полностью — пиши автоматически по порядку
Пример — Углерод C (Z = 6):
2-й жёлтый (p) элемент 2-го периода → внешний уровень 2p²
Конфигурация: 1s² 2s² 2p²
Квантовые ячейки

Правило троллейбуса (правило Хунда)

🚎
Мнемоника Электрон — как пассажир в троллейбусе
Всегда садится на свободное место, если оно есть. Только если мест нет — подсаживается к другому.
N — азот (2p³)
Три ячейки, 3 электрона:
↑ ↑ ↑
3 неспаренных → валентность 3
O — кислород (2p⁴)
Три ячейки, 4 электрона:
↑↓ ↑ ↑
2 неспаренных → валентность 2
Возбуждённое состояние (S*)

Электронная пара распаривается, один электрон переходит на свободную ячейку выше → растёт валентность.

S основное: ...3s² 3p⁴ → 2 неспаренных (валентность 2)
S* первое: ...3s² 3p³ 3d¹ → 4 неспаренных (валентность 4)
S** второе: ...3s¹ 3p³ 3d² → 6 неспаренных (валентность 6)
Важные исключения — выучить!

Аномалии хрома и меди

d-орбиталь особо устойчива, когда заполнена ровно наполовину (d⁵) или полностью (d¹⁰). Поэтому электрон «прыгает» с s на d.

Хром Cr (Z = 24)
Ожидаемо: ...4s²3d⁴
Фактически:
...4s¹ 3d⁵
d⁵ — полузаполненная ✓
Медь Cu (Z = 29)
Ожидаемо: ...4s²3d⁹
Фактически:
...4s¹ 3d¹⁰
d¹⁰ — полностью заполненная ✓
Группы

Главная и побочная подгруппы

Главная подгруппа
Число e⁻ на внешнем уровне = номер группы.

C (IV группа): 2s²2p² → 4 е⁻ ✓
Побочная подгруппа
Внешний уровень — только s-орбиталь с максимальной цифрой.

Cr (VI группа): внешний = 4s¹ (1 е⁻)
Валентные электроны d-элементов = внешняя s + предвнешняя d.
Cr: 4s¹ + 3d⁵ = 6 валентных = номер группы ✓
Ионы

Электронная конфигурация иона

⊖ Анион (M⁻ⁿ)
Добавляем n e⁻ на внешний уровень.
Cl⁻: 1s²2s²2p⁶3s²3p⁶
(добавили 1e⁻ на 3p)
⊕ Катион (M⁺ⁿ)
Убираем n e⁻ с внешнего уровня.
Ca²⁺: 1s²2s²2p⁶3s²3p⁶
(убрали 2e⁻ с 4s)
Лайфхак: ионы со структурой благородного газа — самые устойчивые. Ca²⁺ и Cl⁻ оба имеют конфигурацию аргона Ar.

Ловушки ЕГЭ

Типичные ошибки

❌ Cr: конфигурация ...4s²3d⁴ — НЕТ! Правильно: 4s¹3d⁵ (d⁵ устойчива)
❌ Cu: конфигурация ...4s²3d⁹ — НЕТ! Правильно: 4s¹3d¹⁰ (d¹⁰ устойчива)
❌ Масса Cl = 35,5 → нейтронов 18,5 — НЕТ! Нужен конкретный изотоп: ³⁵Cl → нейтронов 18
❌ У Li и Na одинаковые конфигурации — НЕТ! Они сходные (ns¹), но не одинаковые
❌ Внешний уровень Cr — это 3d⁵ — НЕТ! Внешний уровень — 4s¹ (максимальная цифра)
Главная последовательность

Порядок заполнения и аномалии

1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p
Cr (Z = 24)
...4s¹ 3d⁵
не 4s²3d⁴ !
Cu (Z = 29)
...4s¹ 3d¹⁰
не 4s²3d⁹ !
Причина: d⁵ (полузаполненная) и d¹⁰ (полностью заполненная) — особо устойчивые конфигурации.
